# Convert2Pdf
Convert different file types to PDF using 3rd party components. It doesn't require Native applications like (MS Office, Acrobat, etc.)


File types supported:

* .doc
* .docx
* .docm
* .rtf
  
* .xls
* .xlsx
  
* .ppt
* .pptx
  
* .msg
  
* .htm
* .html
  
* .txt
  
* .tif
* .jpg
* .png

Components Used:

* Aspose.Words
* Aspose.Cells
* Spire.Presentation
* MsgReader - https://github.com/Sicos1977/MSGReader
